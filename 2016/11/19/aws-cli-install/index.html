インストールスクリプト

pip のウェブサイトからインストールスクリプトをダウンロードし実行します。

4
5
6

$ curl "https://bootstrap.pypa.io/get-pip.py" -o "get-pip.py"

$ pip -V
pip 8.1.2 from /usr/local/lib/python2.7/site-packages (python 2.7)

$ sudo python get-pip.py
