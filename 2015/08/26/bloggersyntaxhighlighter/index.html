サンプル

手順通り入れるとこんな感じ？

phpだと、**class=“brush:php;”**を指定すればOK

<pre class="brush:php;"></pre>

囲むと下のように表示されます。

<?php
    $example = range(0, 9);
    foreach ($example as $value) {
      echo $value;
    }

簡単でしたね。
