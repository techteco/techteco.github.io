2. node.jsをインストール

  • 公式サイト node.js

最新バージョンをインストール

バージョンは、latestを指定することで最新版をインストールする。 インストール後は、lsでバージョン確認

$ nodebrew install latest
$ nodebrew ls
v0.12.7

current: none
$

currentの指定

インストールしただけだと、current: noneとなっているように、使えません

$ nodebrew use v0.12.7
use v0.12.7
$ node -v
v0.12.7
$
